Pygame Physics & Color Animation
PythonとPygameライブラリを使用した、2D物理アニメーションのデモンストレーションです。
3つの異なる運動特性を持つオブジェクトを通じて、基本的な座標計算と物理シミュレーションを表現しています。

🎬 デモンストレーション内容
このプログラムでは、以下の3つの異なる挙動を持つ円を描画します。

水平往復運動 (Circle 1): 左右の壁で跳ね返る、基本的な1次元の等速直線運動。

斜め往復運動 (Circle 2): 画面の上下左右の境界で跳ね返る、2次元の等速直線運動。

重力シミュレーション (Circle 3): 加速度（accel）と速度ベクトルを用いた、放物線運動と地面でのバウンド。

すべてのオブジェクトは、フレームごとにRGB値が独立して計算され、サイケデリックに色が変化し続ける視覚効果を備えています。

🛠 技術的なハイライト
物理計算:

等速運動: x += speed * direction によるシンプルな位置更新。

加速度運動: velocity += accel -> position += velocity という、物理エンジンで一般的に使われる2段階更新アルゴリズムの実装。

境界判定: オブジェクトの半径（radius）を考慮した正確な壁の衝突判定と反転処理。

カラーアルゴリズム: 周期の異なる素数的な増分値を用いて色を更新することで、常に変化し続けるグラデーションを実現。

🚀 実行方法
Python環境とPygameライブラリが必要です。

Bash
# Pygameのインストール
pip install pygame

# プログラムの実行
python main.py
🕹 コントロール
終了: ウィンドウの「×」ボタンをクリックしてプログラムを停止します。

📂 ソースコードの構成
pygame.init(): システムの初期化。

while running:: 60FPSで制御されたメインゲームループ。

pygame.display.flip(): ダブルバッファリングによる滑らかな画面更新。
